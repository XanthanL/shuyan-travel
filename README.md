# 树言·旅记 | Shuyan's Travelogue

> “既然这太阳总会准时升起，那我们就没必要急着奔跑。”

## ✦ 简介

这是一个关于行走、触碰与记录的数字日记本。

从 2018 年那个走出北京格子间的清晨开始，我用了八年时间，在大兴安岭的积雪、河西走廊的暴雨、喀什的茶馆与拉萨的暖阳间，打捞那些极其具体的、带有体温的真实。

这个网站不是为了歌颂宏大的意义，而是为了记住：生活本身就是一种由汗水、石头和风干肉堆砌而成的、万劫不磨的真实。我把这些被时间磨薄的瞬间记下来，作为我这八年路的一个交待。

## ✦ 核心逻辑

本项目的设计与叙事遵循以下三条底线逻辑：
- **物质真实**：拒绝形而上的浪漫化。记录老茧的厚度、墙面的裂痕与空气中干燥的木头味道。
- **尊重缓慢**：反对被“速度”异化。时间不以 KPI 计算，而以铜壶的烟火、夕阳的长度与马铃的余响计算。
- **行走扎根**：文字不应悬浮。只有双脚被砂石磨出的痛觉，才是通往真相的唯一路标。

## ✦ 站点结构

- **时间线 (`index.html`)**：八年行走的沉淀，按时间顺序排列的故事索引。
- **旅行地图 (`map.html`)**：交互式地理归宿，利用 Leaflet 连接故事与土地的精准坐标。
- **故事空间 (`story.html`)**：深度叙事环境，采用“纸墨感”排版与首字下沉设计。
- **关于树言 (`about.html`)**：一个大地叙事者的行走逻辑与行囊信条。

## ✦ 技术特性

- **视觉风格**：经典文学报刊排版风格，大地色系（Earth Tones），追求极简与纸质感。
- **核心架构**：原生 HTML5 / CSS3 / JavaScript (ES6+)，无繁重框架，回归静态阅读。
- **地图驱动**：集成 Leaflet.js 交互式地图库，支持地理坐标与叙事内容的双向联动。
- **数据管理**：故事内容与地理数据通过 `js/stories.js` 与 `js/map.js` 进行模块化常量维护。
