# 树言·旅记

我是树言。

2018年的那个清晨，我从北京的一间格子间里走出来，丢掉了一份体面的合同，买了一张北上的硬座票。那时候我还没想清楚自已要去哪儿，只觉得脚底下的柏油马路太烫，烫得我那点关于“自我”的幻象快要化了。

这八年，我一直在走。

我走过大兴安岭没过膝盖的大雪，走过河西走廊泼天而来的暴雨，在喀什的茶馆里听过打铁的节奏，也在拉萨的阳光里嚼过干硬的奶渣。我不是去旅行的，我是在去“找自已”的路上，不小心看清了大地。

这个网站，是我这八年路的一个交待。

这里没那么多宏大的意义。这里只有一些被磨薄的鞋底、一些被盐巴杀透了的风干肉，和一些在极寒或极热中，依然拼尽全力活得极其具体的人。我把他们记下来，不是为了歌颂，而是为了记住：生活本身就是一种由汗水和石头堆砌而成的、万劫不磨的真实。

如果你路过这里，别急着走。找个地方坐下，看一看那些被阳光照亮的浮尘，听一听那串消失在柏油路尽头的铃声。

路还在前头。既然这太阳总会准时升起，那我们就没必要急着奔跑。

---

**故事还在继续，路也还在脚下。**
**2026年 · 树言**
